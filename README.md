# desafio-tela-de-login

- Prompt IA:
  
  ```
  Aja como um especialista em desenvolvimento web acessível e moderno. 
  Sua tarefa é criar o código completo de uma tela de login responsiva,
  usando apenas HTML semântico, CSS moderno via TailwindCSS, e DaisyUI via CDN, 
  sem nenhuma linguagem de programação (nada de JavaScript).
    
  Especificações obrigatórias:
    
  Use apenas HTML e utilize meta tags corretas para estrutura e SEO.
    
  Implemente acessibilidade seguindo o padrão ARIA.
    
  O tema inicial deve ser escuro, com possibilidade de alternar para o 
  claro usando as classes do DaisyUI.
    
  O layout deve ser totalmente responsivo, funcionando bem em dispositivos 
  móveis e desktop.
    
  Os campos do formulário devem ser: e-mail, senha, lembre-me (checkbox), 
  link "esqueci a senha", botão de login e ícones ao lado dos inputs (por exemplo, 
  envelope no campo de e-mail e cadeado no campo de senha).
    
  Inclua um nome de sistema fictício e estilizado como cabeçalho 
  (você pode criar um nome criativo que combine com o layout).
    
  Todo o conteúdo textual deve estar em português.
    
  O código deve conter apenas o que foi solicitado acima, sem interações JS ou scripts adicionais.
